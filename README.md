# HR Analytics: Employee Attrition Analysis

Analyzed 1470+ employee records to identify key factors driving attrition using Python.

**Key Insights:**
- Age group 18-25 shows highest attrition rate
- Sales department has the highest attrition % 
- Employees who left had 25% lower avg salary than those who stayed

**Tools:** Python, Pandas, Matplotlib, Data Visualization, Business Analytics
**Business Impact:** Helps HR reduce hiring costs by identifying at-risk employees
